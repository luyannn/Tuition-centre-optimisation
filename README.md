# Tuition-centre-optimisation
## This repository contains integer programming solutions for assigning tutors to students under given scenarios:
### Scenario A: To minimize total number of tutors assigned while maximizing tutor’s preference on tuition centre
### Scenario B: Balance tutor’s workload while maximizing tutor’s preference on the tuition centre
### It also includes visualizations of the tutor workload, preference match breakdown and the assignment heatmap to summarize and display the validity and accuracy of the optimization done by the model
